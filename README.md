实现功能：

1，根据客户二维码规则，识别输入码并自动生成新码

2，Sqlite本地数据保存下载

3，大族激光打标机通讯协议的实现，自动生成二维码并打标

4，打标信息通过MES系统API接口上传

5，产品图号管理，打标机，MES网络配置 管理

6，自定义配置
